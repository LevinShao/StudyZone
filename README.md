# **StudyZone**

![Python](https://img.shields.io/badge/python-3.14-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

Welcome to StudyZone! StudyZone is an all-in-one productivity application built entirely in Python using Tkinter. It is designed to help users organise their work, improve study habits, boost their confidence, and maintain a healthy balance between productivity and wellbeing. StudyZone combines a wide variety of study, organisation, and self-improvement tools into a single platform, hence providing a convenient all-in-one workspace for managing everyday tasks and building personal growth.

# **Core Features**

- ✅ Task Tracker ✅
- 🎯 Goal Tracker 🎯
- 🔁 Habit Tracker 🔁
- 📖 Digital Notebook 📖
- ⏰ Pomodoro Timer ⏰
- 🧠 Flashcard System 🧠
- 📅 Calendar & Reminders 📅
- 🔥 Login Streak Tracking 🔥
- 🎵 Built-in Music Player 🎵
- ...and many more.

# **Security Features**

- SHA-256 password hashing
- Registration input validation
- Login authentication
- Login rate limiting
- Duplicate username detection
- Secure JSON loading with corruption handling
- Automatic backup of user databases
- Data recovery from backup files

# **Technologies Used**

- Tkinter

- Pygame
- JSON
- hashlib

- Regular Expressions (Regex)

# **Installation Procedure**

Clone the repository

```bash
git clone https://github.com/LevinShao/StudyZone.git
```

Move into the project

```bash
cd StudyZone
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python main.py
```
