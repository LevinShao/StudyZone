# **StudyZone**
![Python](https://img.shields.io/badge/python-3.14-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

Welcome to StudyZone. This project aims to help you with all of your struggles in life. Although it may not be perfect, it still has decent functionality, or at least I hope so.

## **Features**
- Task Tracker, Goal Tracker, Habit Tracker
- Digital Notebook & Flashcards
- Calendar & Reminders
- Pomodoro Timer
- Skill Training Menu
- Express Yourself Menu (Will be reworked in the future)
- File Storage (Currently doesn't work too well)
- Music System
- Streak System
